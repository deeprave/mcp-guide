# Commit Policy: Minimal

A subject line that describes the change. No format rules beyond that.

## Rules

- Write a clear, concise subject line
- Add a body if it helps the reader understand the change
- No enforced format, type prefix, or length limit

## Rationale

Appropriate for personal projects, prototypes, or teams that do not rely on
commit message tooling and prefer not to enforce a format.
